# Zinger-s_method

В программе реализовано нахождение звездного времени, при котором зенитные расстояния двух заданных звезд равны для наблюдателя на заданной географической широте методами бисекции и Ньютона. Сначала считаем начальное приближение, которое используется для двух методов, осуществляется поиск отрезка на интервале [0,2п] на концах которого функция имеет разные знаки (следсвие из теоремы Больцмана-Коши) и берем его сердину. Метод бисекции заключается в том, что на каждой итерации мы делим найденный отрезок попалам и вычисляем значение функции в этой точке.Потом проверяем, с каким из концов исходного отрезка значение F имеет одинаковый знак и продолжаем итерации с уменьшинным вдвоем отрезком, пока длина отрезка не станет меньше удвоенной машинной эпсилон, умноженной на модуль середины, либо пока значение функции в середине не окажется равным нулю с учетом погрешности. На каждой итерации выводим промежуточные значения. Метод бисекции имеет линейную сходимость, не требует вычисления производной и не может разойтись. Для метода Ньютона была заранее вычислена первая производная от уравнения. Метод заключается в том, чтобы на каждой итерации вычислять текущее значение функции и ее производной, определять поправку (значение функция разделенную на значение производной) и вычислять значение корня вычитая эту поправку, т.е. геометрически это значит что мы в текущей точке проводим касательную к графику функции, и следующим приближением будет точка пересечения этой касательной с осью абсцисс. Так же выводим промежуточные значения и итерации продолжаются, пока абсолютная величина поправки не станет меньше или равна машинной эпсилон, умноженной на модуль нового приближения и сложенной с машинной эпсилон, либо не будет достигнуто максимальное число итераций (50). Метод Ньютона имеет квадратичную сходимость, следовательно требует меньшего числа итераций, но может разойтись (в случае если производная будет равна нулю).
